 🌐 TalkTranslate — Language Translation Tool

A desktop application that translates text between multiple languages, built as part of the **CodeAlpha Artificial Intelligence Internship** (Task 1).

✨ Features
- Translate text between English, Hindi, French, Spanish, and German
- Simple, modern light-themed user interface
- One-click copy of translated text to clipboard

 🛠 Built With
- **Python 3**
- **Tkinter** — for the graphical user interface
- **deep-translator** — for connecting to Google Translate

📦 How to Run

1. Clone this repository:https://github.com/Paraszzz/CodeAlpha_LanguageTranslator.git
2. Install the required library:pip install deep-translator
3.  Run the app:python translator_app.py
 
 �🖥 How to Use
1. Type the text you want to translate into the input box.
2. Select the source language ("From") and target language ("To").
3. Click **Translate**.
4. The result appears below — click **Copy to Clipboard** to copy it.
